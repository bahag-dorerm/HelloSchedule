# Convert CSV to JSON

## Prerequisites
* You need node `^14`. Best use nvm.

## One Time Setup
* `npm i`

## Script Commands
* Test Driven Development: `npm run tdd`
* Linting `npm run lint`
* Clean Up Build Artifacts `npm run clean`
* Build `npm run build`
* Test the Build `npm run test` (you need build before!)
* Deploy `npm run deploy`
